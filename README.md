# 🚀 SynergyConnect — Университетская социальная экосистема

**SynergyConnect** — это платформа для студентов, преподавателей и работодателей, которая объединяет:

- 📢 **Социальную сеть** с постами, комментариями и группами.
- 📚 **Учебные группы** с иерархией: Университет → Факультет → Кафедра → Группа.
- 📌 **Рабочие доски** со стикерами, задачами и умными напоминаниями.
- 💼 **Карьерный портал** с вакансиями, стажировками и прямым чатом с работодателями.
- 🔐 **2FA** и шифрование данных для безопасности.
- 📱 **Мобильное приложение** (в планах).

## 🎯 Цель проекта

Создать единую экосистему для университета, где:
- **Студенты** получают информацию, создают проекты и находят работу.
- **Менторы** быстро доносят информацию до групп.
- **Работодатели** находят таланты и общаются напрямую с кандидатами.

## 🛠 Технологический стек

| Компонент | Технология |
|---|---|
| Backend | Go 1.21+, Gin |
| База данных | PostgreSQL 15 |
| Кэш | Redis 7 |
| Фронтенд | React + TypeScript + TailwindCSS |
| Контейнеризация | Docker + Docker Compose |
| CI/CD | GitHub Actions |
| Оркестрация | Kubernetes |

## 🏗 Архитектура

Проект построен на принципах **Clean Architecture** и **Domain-Driven Design**.
backend/
├── cmd/api/ # Точка входа
├── internal/
│ ├── domain/ # Бизнес-сущности
│ ├── application/ # Use Cases
│ ├── infrastructure/# База данных, Redis, Email
│ └── interfaces/ # HTTP-обработчики
└── pkg/ # Переиспользуемые утилиты
